# Hospital-Management-System
Technologies Used in Hospital Management System:

For Frontend: HTML, CSS, BootStrap, JavaScript

For Backend: C#, ASP.NET

For Database: SQL

Features of Hospital Management System:
1. Patient:

1. Patient Home
2. Current Appointment
3. Bills History
4. Treatment History
5. Take Appointment
6. Notifications
7. Feedback

2. Doctor:

1. Doctor Profile
2. Pending Appointments
3. Todays Appointmemts
4. History Update
5. Generate Bill
6. Patient History

3. Administrator:

1. Admin Home
2. View Doctors
3. View Patients
4. View Other Staff
5. Search Box
6. Add/Remove
